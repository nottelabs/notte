Notte allows you to run browser sessions in different geographic regions by using residential proxies with location targeting.

## How It Works

When you enable proxies with geographic targeting, your session will route through residential IPs in the specified region:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import NotteProxy, ProxyGeolocation, ProxyGeolocationCountry

  client = NotteClient()

  # Session in United States
  proxy = NotteProxy(
      geolocation=ProxyGeolocation(
          country=ProxyGeolocationCountry("US")
      )
  )

  with client.Session(proxies=[proxy]) as session:
      page = session.page
      page.goto("https://example.com")
      # Traffic appears to come from the US
  ```
</CodeGroup>

## Supported Regions

Notte supports residential proxies in multiple countries. Common regions include:

- **North America**: US, CA, MX
- **Europe**: GB, DE, FR, ES, IT, NL
- **Asia Pacific**: JP, AU, SG, IN
- **Latin America**: BR, AR, CL
- **Middle East**: AE, IL

<Note>
  Contact support for specific country availability and custom region requirements.
</Note>

## Use Cases

### 1. Geo-Restricted Content

Access content that's only available in specific regions:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import NotteProxy, ProxyGeolocation, ProxyGeolocationCountry

  client = NotteClient()

  # Access UK-only content
  uk_proxy = NotteProxy(
      geolocation=ProxyGeolocation(country=ProxyGeolocationCountry("GB"))
  )

  with client.Session(proxies=[uk_proxy]) as session:
      page = session.page
      page.goto("https://bbc.co.uk/iplayer")
  ```
</CodeGroup>

### 2. Price Comparison

Check prices across different regions:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import NotteProxy, ProxyGeolocation, ProxyGeolocationCountry

  client = NotteClient()

  regions = ["US", "GB", "DE", "JP"]
  prices = {}

  for region in regions:
      proxy = NotteProxy(
          geolocation=ProxyGeolocation(country=ProxyGeolocationCountry(region))
      )

      with client.Session(proxies=[proxy]) as session:
          page = session.page
          page.goto("https://example.com/product/123")

          price = page.locator(".price").inner_text()
          prices[region] = price

  print(f"Prices by region: {prices}")
  ```
</CodeGroup>

### 3. Regional Testing

Test your application from different locations:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import NotteProxy, ProxyGeolocation, ProxyGeolocationCountry

  client = NotteClient()

  test_regions = ["US", "EU", "APAC"]

  for region in test_regions:
      proxy = NotteProxy(
          geolocation=ProxyGeolocation(country=ProxyGeolocationCountry(region))
      )

      with client.Session(proxies=[proxy]) as session:
          page = session.page
          page.goto("https://your-app.com")

          # Test region-specific features
          element = page.locator(f"[data-region='{region}']")
          assert element.is_visible()
  ```
</CodeGroup>

### 4. Load Distribution

Distribute scraping load across multiple regions:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import NotteProxy, ProxyGeolocation, ProxyGeolocationCountry
  import random

  client = NotteClient()

  regions = ["US", "CA", "GB", "DE", "FR"]

  for url in urls_to_scrape:
      # Randomly select a region
      region = random.choice(regions)
      proxy = NotteProxy(
          geolocation=ProxyGeolocation(country=ProxyGeolocationCountry(region))
      )

      with client.Session(proxies=[proxy]) as session:
          page = session.page
          page.goto(url)
          # Scrape data
  ```
</CodeGroup>

## Region + Stealth

Combine region targeting with stealth features for maximum effectiveness:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import NotteProxy, ProxyGeolocation, ProxyGeolocationCountry

  client = NotteClient()

  proxy = NotteProxy(
      geolocation=ProxyGeolocation(country=ProxyGeolocationCountry("US"))
  )

  with client.Session(
      proxies=[proxy],
      browser_type="chrome",
      viewport_width=1920,
      viewport_height=1080
  ) as session:
      page = session.page
      page.goto("https://example.com")
  ```
</CodeGroup>

## Default Proxies (No Region)

If you don't specify a region, Notte uses default residential proxies:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  # Default proxies (region selected automatically)
  with client.Session(proxies=True) as session:
      page = session.page
      page.goto("https://example.com")
  ```
</CodeGroup>

## Best Practices

### 1. Match Locale to Region

Set appropriate language and timezone for the target region:

```python
# For UK sessions
proxy = NotteProxy(
    geolocation=ProxyGeolocation(country=ProxyGeolocationCountry("GB"))
)

with client.Session(
    proxies=[proxy],
    user_agent="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
) as session:
    # UK locale automatically matched
    pass
```

### 2. Respect Rate Limits

Don't abuse region switching to bypass rate limits:

```python
import time

for region in regions:
    # Add delay between regions
    time.sleep(5)

    proxy = NotteProxy(
        geolocation=ProxyGeolocation(country=ProxyGeolocationCountry(region))
    )
    # ... automation
```

### 3. Cache Region-Specific Data

Store results by region to avoid redundant requests:

```python
import json
from pathlib import Path

cache_file = Path("regional_cache.json")

# Load cache
if cache_file.exists():
    cache = json.loads(cache_file.read_text())
else:
    cache = {}

# Check cache before making request
if region not in cache:
    # Fetch data for region
    cache[region] = fetch_data(region)
    cache_file.write_text(json.dumps(cache))
```

## External Proxy Providers

You can also use external proxy providers with region support:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.types import ExternalProxy

  client = NotteClient()

  # Use your own proxy with region
  proxy = ExternalProxy(
      server="http://us-proxy.example.com:8080",
      username="user",
      password="pass"
  )

  with client.Session(proxies=[proxy]) as session:
      page = session.page
      page.goto("https://example.com")
  ```
</CodeGroup>
