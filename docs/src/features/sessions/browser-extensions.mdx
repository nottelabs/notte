Browser extensions can enhance your sessions with additional capabilities like ad blocking, privacy tools, and custom functionality.

<Warning>
  Browser extension support is currently limited. Contact support for specific extension requirements.
</Warning>

## Using Custom Chrome Arguments

Configure browser behavior with Chrome arguments:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  with client.Session(
      chrome_args=[
          "--disable-blink-features=AutomationControlled",
          "--disable-dev-shm-usage",
          "--no-sandbox"
      ]
  ) as session:
      page = session.page
      page.goto("https://example.com")
  ```
</CodeGroup>

## Common Chrome Arguments

### Performance Optimization

```python
performance_args = [
    "--disable-dev-shm-usage",      # Overcome limited resource problems
    "--disable-gpu",                 # Disable GPU hardware acceleration
    "--no-sandbox",                  # Disable sandbox (use with caution)
    "--disable-setuid-sandbox"       # Disable setuid sandbox
]

with client.Session(chrome_args=performance_args) as session:
    pass
```

### Privacy and Stealth

```python
privacy_args = [
    "--disable-blink-features=AutomationControlled",  # Hide automation
    "--disable-web-security",                          # Disable CORS (testing only)
    "--disable-features=IsolateOrigins,site-per-process"  # Disable site isolation
]

with client.Session(chrome_args=privacy_args) as session:
    pass
```

### Window and Display

```python
display_args = [
    "--window-size=1920,1080",      # Set window size
    "--start-maximized",             # Start maximized
    "--disable-infobars",            # Disable info bars
    "--disable-extensions"           # Disable all extensions
]

with client.Session(chrome_args=display_args) as session:
    pass
```

## Ad Blocking (Without Extensions)

Block ads using request interception instead of extensions:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  with client.Session() as session:
      page = session.page

      # Block ad domains
      blocked_domains = [
          "doubleclick.net",
          "googlesyndication.com",
          "googletagmanager.com"
      ]

      def block_ads(route):
          if any(domain in route.request.url for domain in blocked_domains):
              route.abort()
          else:
              route.continue_()

      page.route("**/*", block_ads)
      page.goto("https://example.com")
  ```
</CodeGroup>

<Tip>
  See the [Ad Blocking](/features/sessions/ad-blocking) guide for comprehensive ad blocking strategies without extensions.
</Tip>

## Privacy Tools (Without Extensions)

Implement privacy features programmatically:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  with client.Session(
      chrome_args=[
          "--disable-blink-features=AutomationControlled",
          "--disable-web-security"
      ]
  ) as session:
      page = session.page

      # Block tracking scripts
      page.route("**/analytics/**", lambda route: route.abort())
      page.route("**/tracking/**", lambda route: route.abort())

      # Remove tracking cookies
      page.context.clear_cookies()

      page.goto("https://example.com")
  ```
</CodeGroup>

## User Scripts

Execute custom JavaScript on page load (similar to userscripts):

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  with client.Session() as session:
      page = session.page

      # Add script that runs on every navigation
      page.add_init_script("""
          // Remove all ads
          setInterval(() => {
              document.querySelectorAll('.ad, .advertisement').forEach(el => el.remove());
          }, 1000);

          // Block specific functions
          window.open = function() { return null; };
      """)

      page.goto("https://example.com")
  ```
</CodeGroup>

## Custom Functionality

Add custom browser behavior:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  with client.Session() as session:
      page = session.page

      # Inject custom CSS
      page.add_style_tag(content="""
          .banner-ad { display: none !important; }
          .popup { display: none !important; }
      """)

      # Inject custom JavaScript
      page.add_script_tag(content="""
          console.log('Custom script loaded');

          // Add custom functionality
          window.customFeature = function() {
              console.log('Custom feature executed');
          };
      """)

      page.goto("https://example.com")
  ```
</CodeGroup>

## Alternative: Use External Browser with Extensions

For full extension support, use an external browser provider with extensions pre-installed:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  # Connect to external browser that has extensions
  cdp_url = "ws://external-browser-with-extensions:9222"

  with client.Session(cdp_url=cdp_url) as session:
      page = session.page
      page.goto("https://example.com")
      # Extensions are active
  ```
</CodeGroup>

<Note>
  See the [External Providers](/features/sessions/external-providers) guide for connecting to browsers with custom configurations.
</Note>

## Extension Alternatives

Instead of browser extensions, consider these alternatives:

### 1. Request Interception (Ad Blocking)

```python
page.route("**/ads/**", lambda route: route.abort())
```

### 2. Script Injection (Custom Functionality)

```python
page.add_init_script("/* custom code */")
```

### 3. Style Injection (UI Modifications)

```python
page.add_style_tag(content="/* custom CSS */")
```

### 4. Proxy-Level Blocking (Network-Wide)

```python
# Use proxy with built-in ad blocking
with client.Session(proxies=True) as session:
    pass
```

## Best Practices

### 1. Prefer Native Solutions

Use Playwright features instead of extensions:

```python
# Good: Native Playwright features
page.route("**/ads/**", lambda route: route.abort())
page.add_init_script("/* custom code */")

# Avoid: Relying on extensions
# (Extensions not fully supported)
```

### 2. Test Chrome Arguments

Not all Chrome arguments work in all contexts:

```python
test_args = [
    "--disable-gpu",
    "--no-sandbox"
]

try:
    with client.Session(chrome_args=test_args) as session:
        page = session.page
        page.goto("https://example.com")
        print("Arguments work!")
except Exception as e:
    print(f"Arguments failed: {e}")
```

### 3. Document Custom Configurations

Keep track of Chrome arguments and their purposes:

```python
CHROME_ARGS = {
    "stealth": ["--disable-blink-features=AutomationControlled"],
    "performance": ["--disable-dev-shm-usage", "--disable-gpu"],
    "testing": ["--disable-web-security", "--no-sandbox"]
}

# Use documented configuration
with client.Session(chrome_args=CHROME_ARGS["stealth"]) as session:
    pass
```

## Limitations

### What Works:
- ✅ Chrome arguments
- ✅ Script injection
- ✅ Style injection
- ✅ Request interception
- ✅ External browsers with extensions

### What Doesn't Work:
- ❌ Installing extensions directly in Notte sessions
- ❌ Chrome Web Store extension installation
- ❌ Extension-specific APIs
- ❌ Extension storage/sync

## Future Support

Browser extension support is planned for future releases. Contact support if you have specific extension requirements for your use case.
