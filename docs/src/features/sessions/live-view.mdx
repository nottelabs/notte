Live View allows you to watch your browser sessions in real-time as they execute, making it easy to debug and monitor automation.

## Quick Start

Open a live viewer automatically by setting `headless=False`:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  # Live viewer opens automatically
  with client.Session(headless=False) as session:
      page = session.page
      page.goto("https://example.com")
      page.click("button.submit")
      # Watch it happen in your browser!
  ```
</CodeGroup>

## Manual Live View

Open the live viewer manually for headless sessions:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  with client.Session(headless=True) as session:
      # Start automation
      page = session.page
      page.goto("https://example.com")

      # Open live viewer
      session.viewer()

      # Continue automation while watching
      page.click("button.submit")
  ```
</CodeGroup>

## Viewer Types

Notte supports multiple viewer types:

### Browser Viewer (Default)

Frame-by-frame replay in your browser:

<CodeGroup>
  ```python Python

      page = session.page
      page.goto("https://example.com")

      # Opens browser viewer
      session.viewer_browser()
  ```
</CodeGroup>

<CodeGroup>

## Live Debugging

Use live view to debug automation issues:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  import time

  client = NotteClient()

  with client.Session(headless=False) as session:
      page = session.page

      # Watch the automation
      page.goto("https://example.com/complex-form")

      # Add pauses to observe
      time.sleep(2)

      page.fill('input[name="email"]', "user@example.com")
      time.sleep(1)

      page.click('button[type="submit"]')
      time.sleep(2)

      # See what happened
  ```
</CodeGroup>

## Monitoring Multiple Sessions

Monitor several sessions simultaneously:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  import time

  client = NotteClient()

  sessions = []

  # Start multiple sessions
  for i in range(3):
      session = client.Session()
      session.start()
      sessions.append(session)

      # Open viewer for each
      session.viewer()

      # Start automation
      page = session.page
      page.goto(f"https://example.com/page{i}")

  # Let them run
  time.sleep(30)

  # Clean up
  for session in sessions:
      session.stop()
  ```
</CodeGroup>

## Live View for Agents

Watch AI agents make decisions in real-time:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  # Watch the agent work
  with client.Session(headless=False) as session:
      agent = client.Agent(session=session, max_steps=10)

      result = agent.run(
          task="Find the pricing page and extract all plan prices"
      )

      print(f"Agent completed: {result.answer}")
  ```

## Live View Best Practices

### 1. Use for Development, Not Production

Live view is great for development and debugging:

```python
import os

# Only use live view in development
is_dev = os.getenv("ENV") == "development"

with client.Session(headless=not is_dev) as session:
    # Live view in dev, headless in prod
    pass
```

### 2. Combine with Pauses

Add strategic pauses to observe behavior:

```python
import time

with client.Session(headless=False) as session:
    page = session.page

    page.goto("https://example.com")
    time.sleep(2)  # Pause to observe

    page.click("button.load-more")
    time.sleep(3)  # Watch content load

    page.screenshot(path="result.png")
```

### 3. Use for Complex Workflows

Live view is especially useful for complex automations:

```python
with client.Session(headless=False) as session:
    page = session.page

    # Complex multi-step workflow
    page.goto("https://example.com/wizard")

    # Step 1
    page.fill('input[name="step1"]', "value1")
    page.click('button.next')

    # Step 2
    page.fill('input[name="step2"]', "value2")
    page.click('button.next')

    # Watch each step execute
```

### 4. Record While Viewing

Combine live view with recording:

```python
with client.Session(headless=False) as session:
    # Watch it live
    page = session.page
    page.goto("https://example.com")
    # ... automation ...

# Get recording after
replay = session.replay()
replay.save("recording.mp4")
```

## Live View vs Recordings

| Feature | Live View | Recordings |
|---------|-----------|------------|
| **Timing** | Real-time during execution | After session ends |
| **Interactivity** | Can pause/observe | Playback only |
| **Use Case** | Debugging, monitoring | Archiving, sharing |
| **Performance Impact** | Minimal | None |
| **Availability** | During session | 24 hours after |

## Viewer Configuration

Set default viewer type:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient
  from notte_sdk.endpoints.sessions import SessionViewerType

  # Set default viewer to CDP
  client = NotteClient(viewer_type=SessionViewerType.CDP)

  with client.Session() as session:
      session.viewer()  # Opens CDP viewer by default
  ```
</CodeGroup>

## Troubleshooting

### Viewer Won't Open

If the viewer doesn't open automatically:

```python
import webbrowser

with client.Session() as session:
    debug_info = session.debug_info()

    # Manually open viewer
    webbrowser.open(debug_info.debug_url)
```

### Multiple Viewers

You can open multiple viewer types simultaneously:

```python
with client.Session() as session:
    page = session.page
    page.goto("https://example.com")

    # Open both viewers
    session.viewer_browser()  # Frame-by-frame
    session.viewer_cdp()      # DevTools

    # Continue automation
```
