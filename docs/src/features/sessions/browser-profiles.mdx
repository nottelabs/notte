---
title: Browser Profiles
description: Persistent browser profiles with saved state across sessions
---

Browser Profiles allow you to maintain persistent browser state across sessions, including cookies, local storage, cache, and browser settings. Profiles enable seamless continuation of authenticated sessions without re-logging in.

## Key Benefits

- **Persistent authentication** - Stay logged in across sessions
- **Faster automation** - Skip login flows on subsequent runs
- **State preservation** - Maintain local storage, cookies, and preferences
- **Multi-account support** - Manage separate profiles for different users
