The Notte Marketplace is a hub for discovering, sharing, and deploying pre-built browser automations and Functions created by the community and Notte team.

## What is Marketplace?

Marketplace provides:

- **Pre-built automations** for common tasks
- **Community Functions** shared by other users
- **Verified templates** from the Notte team
- **One-click deployment** to your account
- **Customizable solutions** you can modify

## Browsing Marketplace

### Explore Categories

Browse automations by category:

- **Data Extraction** - Web scraping and data collection
- **Testing** - Automated testing Functions
- **Monitoring** - Website monitoring and alerts
- **Social Media** - Social media automation
- **E-commerce** - Product research and price tracking
- **Lead Generation** - Contact and lead collection
- **Account Management** - Account creation and management

### Search Automations

Find specific automations:

1. Go to [console.notte.cc/marketplace](https://console.notte.cc/marketplace)
2. Use the search bar to find automations
3. Filter by category, popularity, or rating
4. Preview automation details

## Installing from Marketplace

### One-Click Install

Deploy automations instantly:

1. Browse Marketplace
2. Click on an automation
3. Review details and requirements
4. Click **Install**
5. Configure parameters
6. Deploy to your account

### Customize Before Installing

Modify automations before deployment:

1. Select an automation
2. Click **Fork & Customize**
3. Edit in Studio or code editor
4. Test your changes
5. Deploy modified version

## Using Installed Automations

### Via Console

Run automations from the console:

1. Go to **My Automations**
2. Select installed automation
3. Click **Run**
4. View results

### Via SDK

Call installed automations from your code:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  # Run marketplace automation
  result = client.marketplace.run(
      automation_id="scrape-product-reviews",
      params={"product_url": "https://example.com/product/123"}
  )

  print(result)
  ```
</CodeGroup>

### Via API

Invoke as API endpoints:

<CodeGroup>
  ```bash cURL
  curl https://api.notte.cc/marketplace/scrape-product-reviews \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"product_url": "https://example.com/product/123"}'
  ```
</CodeGroup>

## Managing Installed Automations

### Update Automations

Keep automations up to date:

1. Go to **My Automations**
2. Check for updates
3. Review changelog
4. Click **Update**

### Configure Settings

Customize automation behavior:

<CodeGroup>
  ```python Python
  from notte_sdk import NotteClient

  client = NotteClient()

  # Configure installed automation
  client.marketplace.configure(
      automation_id="price-monitor",
      settings={
          "check_interval": "1h",
          "alert_email": "user@example.com",
          "price_threshold": 100
      }
  )
  ```
</CodeGroup>

**1. Clear Documentation**
```python

Extracts product prices from major e-commerce sites.

- product_url (str): URL of the product page
- currency (str): Currency code (USD, EUR, etc.)

Returns:
- dict: Product name, price, and availability
"""
```

```python
def scrape_product(product_url: str):

        # Automation logic
```
