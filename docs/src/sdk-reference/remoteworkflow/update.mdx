---
title: "update"
description: "Update the function with a new code version"
---



```python
function = notte.Function("<your-function-id>")
function.update(workflow_path="<path-to-your-function.py>")
```

If you set a version, only that version will be updated


## Parameters

<ParamField path="workflow_path" type="str" required>
</ParamField>

<ParamField path="version" type="UnionType[str, None]" default="None">
</ParamField>

## Returns

`None`
