---
title: "__init__"
description: "Initialize a NotteFunction"
---

Create a new function or load an existing one.

## Parameters

<ParamField path="function_id" type="str | None" default="None">
  The ID of an existing function to load.
</ParamField>

<ParamField path="decryption_key" type="str | None" default="None">
  Key required for local execution and downloading function code.
</ParamField>

<ParamField path="workflow_path" type="str">
  The path to the Python file to upload (required when creating a new function).
</ParamField>

<ParamField path="name" type="str | None">
  Display name for the function.
</ParamField>

<ParamField path="description" type="str | None">
  Description of what the function does.
</ParamField>

<ParamField path="shared" type="bool" default="False">
  Whether the function is publicly accessible.
</ParamField>

## Returns

`None`
